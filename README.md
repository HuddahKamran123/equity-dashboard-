# Employment Equity Gap Dashboard · TBS 2024-25

Interactive dashboard for EDI policy leads at the Treasury Board of Canada Secretariat.

**Question it answers:** Which departments, for which equity groups, are furthest below their workforce availability benchmark — and what kind of review does that signal?

## What it does

- Ranks all 71 federal departments across 4 designated equity groups by representation gap
- Flags priority departments (bottom quartile per group)
- Shows severity scale: Slight / Moderate / Substantial / Severe
- Tracks year-over-year trends for 35 departments present in both fiscal years
- Generates 5-block presentation summaries per department-group combination

## Data

- **Source:** Treasury Board of Canada Secretariat — Employment Equity Demographic Snapshots (2023-24, 2024-25)
- **Enrichment:** Public Service Employee Survey 2024 (PSES)
- **284 rows** embedded in the dashboard (2024-25 fiscal year)

## Governance

This is a signal surface, not a decision engine. Every priority flag requires human review before action. No causal claims. No composite scores. No staffing recommendations.

**Approved review vocabulary:** recruitment pipeline review · retention review · promotion pipeline review · accessibility review · workplace inclusion review
